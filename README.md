# Resume Coach - AI Resume Builder

## Package manager

This repository uses `pnpm`.

### Common commands

```bash
pnpm install
pnpm dev
pnpm test
pnpm test:e2e
pnpm lint
pnpm lint:fix
pnpm build
```

### Playwright

```bash
pnpm exec playwright install --with-deps
pnpm test:e2e
pnpm test:e2e:ui
pnpm exec playwright test --project=chromium
pnpm exec playwright test --debug
pnpm exec playwright codegen
```
